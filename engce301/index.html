ENGCE301

Software Design and Development

หลักสูตรที่จะเปลี่ยนคุณจาก "ผู้เขียนโค้ด" สู่ "วิศวกรซอฟต์แวร์" มืออาชีพ พร้อมสร้าง Full-Stack Web Application ที่ทันสมัยด้วยเทคโนโลยีที่เป็นที่ต้องการของตลาด

เริ่มต้นการเดินทาง

เส้นทางการเรียนรู้และปฏิบัติการ (Labs)

สัปดาห์ที่ 1

Introduction to Software Engineering

ภาพรวมวิศวกรรมซอฟต์แวร์, SDLC, จาก Waterfall สู่ Agile/DevOps และติดตั้งเครื่องมือพัฒนา

ดูรายละเอียด Lab 01

สัปดาห์ที่ 2

Version Control & Collaborative Development

เรียนรู้ Git Workflows, การทำงานร่วมกันเป็นทีมผ่าน GitHub และกระบวนการ Pull Request

ดูรายละเอียด Lab 02

สัปดาห์ที่ 3

Requirements Analysis & User Stories

เทคนิคการเก็บความต้องการ, การเขียน User Stories และ Acceptance Criteria ที่มีประสิทธิภาพ

ดูรายละเอียด Lab 03

สัปดาห์ที่ 4

System Modeling & UML Design

ออกแบบและจำลองระบบด้วย UML Diagrams (Use Case, Class, Sequence)

ดูรายละเอียด Lab 04

สัปดาห์ที่ 5

Software Architecture และ High Level Design ระดับพื้นฐาน

ออกแบบโครงสร้างภาพรวม (High-Level) ของระบบ, กำหนด Components หลักและความสัมพันธ์ระหว่างกัน

ดูรายละเอียด Lab 05

สัปดาห์ที่ 6

Low Level Design ระดับพื้นฐาน โดยใช้ React.js

ออกแบบโครงสร้างภายในของ React Component ให้ดูแลรักษาง่ายและนำกลับมาใช้ใหม่ได้

ดูรายละเอียด Lab 06

สัปดาห์ที่ 7

Frontend Development with Modern JavaScript

เจาะลึก Modern JS (ES6+), DOM, Promises, async/await และการเรียกใช้ API

ดูรายละเอียด Lab 07

สัปดาห์ที่ 8: 📝 สอบกลางภาค

Midterm Examination

ทบทวนเนื้อหา: SE Principles, UML, High-Level Design

สัปดาห์ที่ 9

Introduction to React.js

เริ่มต้นสร้าง Component-based UI ด้วย React, JSX, State & Props และ React Hooks

ดูรายละเอียด Lab 09

สัปดาห์ที่ 10

Advanced React & Component Design

พัฒนา Custom Hooks, ทดสอบ Components และเทคนิคการทำ Performance Optimization

ดูรายละเอียด Lab 10

สัปดาห์ที่ 11

Backend Development with Node.js

สร้าง Server-side application ด้วย Node.js และ Express.js, ออกแบบ RESTful API

ดูรายละเอียด Lab 11

สัปดาห์ที่ 12

Database Integration & Data Management

เชื่อมต่อและจัดการข้อมูลด้วย NoSQL Database (MongoDB) ผ่าน Mongoose ODM

ดูรายละเอียด Lab 12

สัปดาห์ที่ 13

Full-Stack Integration & Authentication

เชื่อม React Frontend กับ Node.js Backend, สร้างระบบยืนยันตัวตนด้วย JWT

ดูรายละเอียด Lab 13

สัปดาห์ที่ 14

Software Testing & Quality Assurance

เรียนรู้กลยุทธ์การทดสอบและลงมือเขียนเทสด้วย Jest และ React Testing Library

ดูรายละเอียด Lab 14

สัปดาห์ที่ 15

DevOps Overview & Deployment Demo

ภาพรวมของ CI/CD และชมการสาธิตการ deploy แอปพลิเคชันขึ้น Cloud (Netlify, Heroku)

ดูรายละเอียด Lab 15

สัปดาห์ที่ 16: 📝 สอบปลายภาค

Final Examination

ทบทวนเนื้อหา: Full-stack development, Testing

สัปดาห์ที่ 17: 🏆 Project Presentation

Final Project Showcase & Wrap-up

นำเสนอโปรเจกต์ที่ทำมาทั้งเทอม, Peer review และเตรียมความพร้อมสู่โลกการทำงานจริง

เครื่องมือและเทคโนโลยี

Frontend

React.js, HTML5, CSS3, ES6+

Backend

Node.js, Express.js

Database

MongoDB, Mongoose

Testing

Jest, React Testing Library

เตรียมตัวให้พร้อมสำหรับความท้าทายที่จะเปลี่ยนอนาคตสายอาชีพของคุณ แล้วพบกันในคลาส!
