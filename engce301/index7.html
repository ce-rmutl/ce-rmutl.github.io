Software Design and Development

หลักสูตรที่จะเปลี่ยนคุณจาก "ผู้เขียนโค้ด" สู่ "วิศวกรซอฟต์แวร์" มืออาชีพ พร้อมสร้าง Full-Stack Web Application ที่ทันสมัยด้วยเทคโนโลยีที่เป็นที่ต้องการของตลาด

ภาพรวมและเป้าหมาย

เข้าใจกระบวนการ

มีความรู้ความเข้าใจเกี่ยวกับวงจรการพัฒนาซอฟต์แวร์สมัยใหม่, การวิเคราะห์ความต้องการ, และการออกแบบเชิงสถาปัตยกรรม

สร้าง Full-Stack App

มีทักษะในการพัฒนาเว็บแอปพลิเคชันแบบ Full-stack ด้วย React, Node.js/Express, SQLite และ MongoDB

ทำงานแบบมืออาชีพ

ส่งเสริมการทำงานเป็นทีม, การใช้เครื่องมือ (Git), การจัดการโครงการ (Agile) และการสื่อสารอย่างมีจริยธรรมวิชาชีพ

Tech Stack

HTML/CSS/JS, React, Node.js, Express, SQLite, MongoDB, Git, Figma, Jest

ผลการเรียนรู้ (CLOs)

CLO1

มีความรู้พื้นฐานด้านวิศวกรรมซอฟต์แวร์, กระบวนการ, และสถาปัตยกรรม

CLO2

วิเคราะห์ปัญหาและความต้องการผู้ใช้ และจัดทำเอกสาร/แบบจำลองความต้องการ

CLO3

ออกแบบสถาปัตยกรรม, ระบบข้อมูล และ UI/UX ให้เหมาะสมกับความต้องการ

CLO4

พัฒนา, ทดสอบ และบูรณาการซอฟต์แวร์ต้นแบบด้วยเทคโนโลยีสมัยใหม่

CLO5

ประเมินและปรับปรุงคุณภาพซอฟต์แวร์ (testing, refactoring, security)

CLO6

ทำงานเป็นทีม, ใช้เครื่องมือ, และสื่อสารทางเทคนิคได้อย่างมีจริยธรรม

แผนการเรียน 17 สัปดาห์

เลื่อนไปทางขวาเพื่อดูสัปดาห์ถัดไป →

Week 1

Intro to SE & SDLC

  • อธิบายบทบาทวิศวกรซอฟต์แวร์
  • เปรียบเทียบ Waterfall vs Agile
  • ติดตั้งเครื่องมือพื้นฐาน

กิจกรรม

  • Workshop ติดตั้ง Git/VSCode/Node.js, สร้าง Git repo, commit HTML แรก
CLO1, CLO4, CLO6

Week 2

Requirements & Semantic HTML

  • ระบุ Stakeholder & requirement
  • ร่างโครง SRS ฉบับแรก
  • สร้างหน้า Landing Page ด้วย HTML5

กิจกรรม

  • Brainstorm requirement, Role-play สัมภาษณ์ผู้ใช้, เริ่มร่าง SRS
CLO1, CLO2, CLO4, CLO6

Week 3

Agile Requirements & CSS

  • เขียน User Story & Acceptance Criteria
  • ออกแบบ Use Case Diagram
  • จัดหน้าเว็บด้วย CSS เบื้องต้น

กิจกรรม

  • แปลง SRS → Product Backlog (Trello), วาด Use Case Diagram, ต่อเติม CSS
CLO2, CLO3, CLO4, CLO6

Week 4

Design Principles & UX/UI

  • อธิบายหลักการออกแบบ (SOLID)
  • สร้าง Personas & User Journey Map
  • ปรับ UI ให้ Responsive (Flexbox/Grid)

กิจกรรม

  • ทำ Personas, ปรับหน้า HTML/CSS ให้เป็น responsive, เตรียมโครงหน้าฟอร์ม
CLO1, CLO3, CLO4, CLO6

Week 5

Prototyping & Basic JS

  • ออกแบบ UI บน Figma (Mockup)
  • อธิบาย Usability Heuristics
  • ใช้ JavaScript (DOM, event, validation)

กิจกรรม

  • ออกแบบหน้าหลักใน Figma, เพิ่ม JavaScript บนฟอร์ม (ตรวจช่องว่าง, validate email)
CLO3, CLO4, CLO6

Week 6

Architecture & Node.js Intro

  • เลือกสถาปัตยกรรม (Client-Server)
  • อธิบายบทบาท backend API
  • สร้าง Express server (อ่าน/เขียน JSON)

กิจกรรม

  • สร้างโปรเจกต์ Node.js/Express, สร้าง route (GET/POST) อ่าน/เขียน `data.json`, ทดสอบด้วย Postman
CLO1, CLO3, CLO4

Week 7

Backend API & SQLite

  • ออกแบบ RESTful API endpoints
  • สร้าง ER Diagram
  • เชื่อม Express API กับ SQLite (CRUD)

กิจกรรม

  • ออกแบบ endpoints, วาด ER Diagram, สร้างฐานข้อมูล SQLite และเชื่อม Express (CRUD)
CLO2, CLO3, CLO4, CLO5

Week 8

สอบกลางภาค (Midterm)

  • ประเมินความเข้าใจ SE, Req, UI/UX, API
  • ประยุกต์ความรู้กับโจทย์ใหม่

กิจกรรม

  • ทฤษฎี (2 ชม.): สอบข้อเขียน
  • ปฏิบัติ (3 ชม.): สอบปฏิบัติ (ออกแบบ Use Case, HTML/CSS, API)
CLO1, CLO2, CLO3, CLO4, CLO6

Week 9

Version Control & Refactor

  • ใช้ Git branching strategy (Git Flow)
  • ทำ Code review พื้นฐาน
  • Refactor โครงสร้าง frontend

กิจกรรม

  • ตั้งค่า branching, แบ่ง feature branch, ทำ PR + review, refactor HTML/CSS/JS
CLO4, CLO5, CLO6

Week 10

Frontend (React)

  • อธิบายแนวคิด React component
  • สร้าง Components (Props, State)
  • เรียกใช้ backend API จาก React

กิจกรรม

  • ตั้งค่า React app (Vite), แปลงหน้า UI เดิมเป็น React components, เชื่อม React กับ Express API
CLO3, CLO4

Week 11

Backend & MongoDB

  • ออกแบบเลเยอร์ backend (controller/service)
  • เปรียบเทียบ RDB vs NoSQL
  • เชื่อม Express กับ MongoDB

กิจกรรม

  • เพิ่ม module ใหม่ในโปรเจกต์ที่ใช้ MongoDB (เช่น user), สร้าง model/schema, เชื่อม React → Express → MongoDB
CLO3, CLO4, CLO5

Week 12

Full-Stack Integration & Security

  • เชื่อมต่อ Full-stack flow (React+Express+DB)
  • ติดตั้ง Authentication (JWT)
  • ใช้ Basic Security (validation, XSS)

กิจกรรม

  • เชื่อมหน้า Login/Register, เพิ่ม middleware ตรวจ JWT, ทำ validation (client+server)
CLO4, CLO5

Week 13

Software Testing

  • อธิบาย Testing Pyramid
  • เขียน Test Case จาก User Story
  • เขียน Unit Test (Jest/React Testing)

กิจกรรม

  • ตั้งค่า Jest & React Testing Library, สร้าง Unit Test สำหรับ function/component, เขียน Integration Test (Supertest)
CLO4, CLO5

Week 14

Quality & Refactoring

  • ระบุ Code Smell
  • ใช้ Static analysis (ESLint)
  • Refactor โค้ดโดยมี Test คุม

กิจกรรม

  • รัน ESLint/Prettier, เลือกส่วนที่มี code smell มา refactor (React/Express), รัน test เพื่อยืนยัน
CLO1, CLO5

Week 15

PM & Deployment (CI/CD)

  • วางแผน Sprint สุดท้าย (Agile PM)
  • อธิบาย CI/CD (Intro)
  • Deploy Full-stack app (Vercel/Render)

กิจกรรม

  • Sprint planning/Retros, ตั้งค่า script build/test, Deploy (Vercel/Render), ทดสอบระบบ deployed
CLO1, CLO4, CLO5, CLO6

Week 16

Final Project Presentation

  • นำเสนอ Term project (Full-stack)
  • อธิบาย Design Decision & Process
  • สะท้อนสิ่งที่เรียนรู้ (Reflection)

กิจกรรม

  • ปฏิบัติ (5 ชม.): Final presentation & live demo, Q&A, Peer evaluation, Retrospective
CLO4, CLO5, CLO6

Week 17

สอบปลายภาค (Final Exam)

  • บูรณาการความรู้ทั้งหมด
  • แก้โจทย์ “Change Request”

กิจกรรม

  • ทฤษฎี (2 ชม.): สอบข้อเขียน (SE process, design, QA, PM)
  • ปฏิบัติ (3 ชม.): สอบปฏิบัติ (แก้/เพิ่ม feature + อธิบายผลกระทบ)
CLO1, CLO2, CLO3, CLO4, CLO5, CLO6
