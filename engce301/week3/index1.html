üìÑ Use Case Scenario Structure

Use Case: Submit Assignment

Actor: Student

Preconditions: Logged in, Before Deadline

Main Flow:

  1. Student goes to assignment page.
  2. System shows upload button.
  3. Student selects file.
  4. System validates file.
  5. System confirms submission.

Alternative Flows:

  • 4a. File too large: Show error, ask to re-upload.
  • 4b. Wrong format: Show error "PDF only".
